Authentication using Neural Network Analysis of Pattern Tracing

Welcome to our Usable Security user study!

As part of this study, you will complete the following steps:

  1. Teach the website how you draw
  2. "Log in" to this website using pattern tracing
  3. Fill out a short questionnaire

This will take approximatly 10 min to complete.
To begin, click here to start the training.

Information

What are we trying to do?
We want to create a way to login to a website, phone, or computer by simply looking at how you trace a shape. In order to do this we need data from lots of people. This is where you come in!
What data are we collecting?
We will be collecting where and when your pointer is while tracing the shape, as well as some basic demographic information in the questionnaire.
